# Action ImportData

| PDF/A version | Conformance level | Class type  | Severity |
| ------------- | ----------------- | ----------  | -------- |
| 1, 2, 3       | bua               | Conformance | 5        |

The PDF/A document _has_ an import data action.

The import data action could be used to populate a form with data from a FDF file.

# Reproducibility
## Adobe Acrobat X Professional
1. Open the properties dialog for any of the following items
- bookmark,
- page thumbnail,
- link,
- media clip,
- form field.

2. switch to Actions tab,

3. select trigger, for example "on Mouse Up",

4. select action "Import Form Data".
