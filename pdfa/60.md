# Font metric

| PDF/A version | Conformance level | Class type  | Severity |
| ------------- | ----------------- | ----------  | -------- |
| 1, 2, 3       | bua               | Conformance | 5        |

The PDF/A document _has_ fonts without metrics.

# Reproducibility
## Edit
Remove from the font descriptor dictionary `/Widths`.
