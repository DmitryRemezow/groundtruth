# XMP PDF/A version identifier

| PDF/A version | Conformance level | Class type  | Severity |
| ------------- | ----------------- | ----------  | -------- |
| 1, 2, 3       | bua               | Conformance | 5        |

The PDF/A document _has not_ <pdfaid:part> and <pdfaid:conformance> in XMP metadata.
