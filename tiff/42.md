# Mandatory TIA tags
The following tags are mandatory in TI-A:

- NewSubfileType
- ImageWidth
- ImageLength
- BitsPerSample
- Compression
- PhotometricInterpretation
- StripOffsets
- Orientation
- SamplesPerPixel
- RowsPerStrip
- StripByteCounts
- PlanarConfiguration

# Reproducibility

Remove any of these tags in a TIFF file.
